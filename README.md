# Inflatable PVC Boat Fender Technical Reference (A / F / D / G / HTM)

A structured technical reference for inflatable PVC boat fenders used in docking protection systems.  
This repository focuses on geometry differences, load distribution behavior, application logic, and procurement considerations for A, F, D, G, and HTM series fenders.

**Last Updated:** 2026-02  
**Maintained by:** Seaman Marine Supplies  

---

## Why This Repository Exists

Selecting the correct inflatable boat fender is not only about size.  
Shape, suspension points, ribbing, and internal structure directly affect:

- Impact absorption
- Rebound control
- Abrasion resistance
- Long-term deformation
- Installation stability

This repository summarizes technical differences between series and provides simplified decision logic for B2B buyers and marina operators.
